# Interactive graph

In this lesson we'll create an interactive graph to solidify our understanding of the hooks we just learned, but also to learn an additional hook. As always, it's a real-world example — you can see a very similar component on Linear's features page.

## Making it interactive

Before we add any motion, we first need to make the graph follow the cursor. Our starting point will be a simple static SVG graph. This exercise will cover what we just learned in the last lesson, but also a few techniques we have learned earlier in the course, and even a new Framer Motion hook.

You can use the hints if you get stuck. And remember, it's okay to struggle, that's how you learn.

## Adding motion

The end result should include not only the motion, but also one detail that makes the graph feel nicer — smooth spring-based transitions as the cursor moves.
