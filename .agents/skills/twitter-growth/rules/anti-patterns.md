# Anti-Patterns

## red flags

| pattern                         | impact                                 | evidence                                                                  |
| ------------------------------- | -------------------------------------- | ------------------------------------------------------------------------- |
| "Comment X for Y" reply farming | halves views, 3x fewer likes           | Ami clean: 1,189K views. Ami "comment ami": 439K views                    |
| Feature-centric language        | caps at <200K views                    | "Automatically save clips" → 36K. "Scan YOUR codebase" → 1,049K           |
| "v0.1.2" update framing         | 5-11x fewer views than "Introducing X" | React Scan launch: 572.8K. Avg update: 51K                                |
| Company account posting         | 316x fewer likes                       | @aidenybai: 3,475 likes. @samedotdev same product: 11 likes               |
| Emojis and hype adjectives      | lower engagement                       | Million 3.0 (emojis): 1,749 likes. React Doctor (no emojis): 10,545 likes |
| Sunday launch                   | worst performing day                   | 2,500 avg likes vs 6,800 on Tuesday                                       |
| No video on product launch      | missing scoring dimension              | `vqv_score` = 0.0 for non-video                                           |
| No npx/CLI CTA                  | low bookmarks                          | With CTA: B/L 1.07. Without: B/L 0.56                                     |
| Missing "open source" closer    | missing trust signal                   | Standard since Nov 2024                                                   |
| Article linked directly         | capped distribution                    | Tweet insight natively, link article in reply                             |

## fixes

- "Tool does X" → "[YOU] do X → [result]"
- "v0.2.0 with features" → "Introducing [New Name]"
- Posted from @company → post from @personal
- "Comment X for Y" → remove, clean launch performs 2.7x better
- No video → add 8-15s demo
- No CTA → add `npx [package]@latest`
